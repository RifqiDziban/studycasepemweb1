  • List dan Image

    Pada dokumen HTML terdapat dua jenis list, yaitu unordered list dan ordered list. Unordered list adalah suatu list yang tidak memperhatikan urutan penomoran sedangkan ordered list adalah list yang memperhatikan urutan penomoran. Untuk membuat suatu unordered list menggunakan tag UL sedangkan untuk membuat ordered list menggunakan tag OL. Dan untuk mengisikan item pada list menggunakan tag LI

    Tag IMG pada dokumen HTML digunakan untuk menambahkan gambar pada suatu halaman web. Untuk menambahkan gambar dapat dilakukan dengan mengubah nilai atribut SRC disesuaikan dengan nama file gambar yang diinginkan. Selain atribut SRC, tag IMG juga memiliki atribut lain seperti ALIGN, BORDER, ALT, HEIGHT, dan WIDTH. Atribut ALIGN digunakan untuk mengatur aligmen. Nilai yang bisa diberikan pada atribut ALIGN adalah TOP, MIDDLE, dan BOTTOM. Atribut BORDER digunakan untuk menentukan ketebalan garis tepi dari suatu gambar. Atribut ALT digunakan untuk memberi keterangan ketika suatu gambar gagal untuk ditampilkan ke web browser. Atribut HEIGHT dan WIDTH digunakan untuk mengatur tinggi dan lebar gambar.
